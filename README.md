# ORACLE-
Oracle is an ESP 32 based truth verification system, that uses multiple sensors to measure stress level and decides not that person is lying. It uses a gsr sweat sensor, detects oxygen saturation of the suspects body, and calculates the probability of that person lying. 


## Why I Made Project ORACLE:
I've always had a fascination for integrating hardware with biometric data and solvoing some kind of problem with it or make something cool!
I got that chance when my school declared the date (which is very soon) for the science fair. So, I am building this project, for the science fair as well as personal interests,
